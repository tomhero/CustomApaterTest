## What's about this app? (เอาภาษาไทยเหอะ)

ก็สามารถแสดง ListView แบบที่ใจเราต้องการได้ (cumtomizable)
และยังสามารถกดเข้าไปดูข้อมูล แล้วแก้ไขได้ด้วย 
จากนั้นก็กด Save แล้วมันจะกลับมาที่หน้า ListView อีกครั้ง
